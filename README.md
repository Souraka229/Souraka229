## `> ./projects.sh`

### 🍽️ **RESTAFY** — Revolutionizing West African Food Tech
> **Status**: Active Founder | **Stage**: Growth  
> **Impact**: Food delivery, POS system, Event ticketing — 0% commission model

**The Problem They Solved:**
- West African restaurants drowning in Excel spreadsheets
- No local payment integration (MTN MoMo, Moov Money, Celtiis Cash)
- Disconnected order management → kitchen chaos

**The Solution:**
- **Centralized Management**: On-premise orders, delivery, QR menu all in one dashboard
- **Real-time Kitchen System (KDS)**: Smart order routing to kitchen staff
- **Local Payment Rails**: Direct integration with Africa's primary mobile money providers
- **Analytics That Matter**: Clear sales tracking, inventory insights, growth metrics
- **For Restaurateurs, by Restaurateurs**: Built to work offline-first, designed for maquis owners

**Tech Stack**: React · Vite · Supabase · Node.js · Mobile Money USSD APIs

**Stats:**
- ✅ MVP launched | ✅ Restaurant partnerships active | ✅ 0% commission commitment live

**Links**: [Website](https://restafy.shop) · [LinkedIn](https://www.linkedin.com/company/restafy)

### 🤖 **ReturnWise** — AI-Powered Migration Support Platform
> **Status**: Competition Winner · IOM Morocco Hackathon  
> **Team**: Second Chance Lab (4 engineers + 1 PM)  
> **Recognition**: OIM Morocco · BENIN AI COMMUNITY · EU Funded Programme

**The Challenge:**
AVRR agents spending hours juggling Excel, documents, and databases to understand one case. Migrants unsure about procedures, required documents, or where their application stands.

**The Solution — Guided by "AI Assists, Humans Decide":**

**For AVRR Agents:**
- Automated vulnerability scoring with explainable results
- AI-recommended reintegration pathways
- Field monitoring dashboards with real-time follow-up
- Contextual AI assistant — information retrieval, not replacement

**For Beneficiaries:**
- Clear guidance portal explaining the return process
- Document preparation checklists
- Real-time application status tracking
- Direct contact support

**Core Principle**: *"ReturnWise structures. Suggests. Compares. Alerts. But it never replaces human judgment in sensitive contexts like migration."*

**Tech Stack**: Python · Machine Learning · React · Node.js · PostgreSQL · Supabase

**My Role**: System Architecture · Data Pipeline · ML Vulnerability Scoring

## `> cat manifesto.txt`

```
The gap between global tech and African business reality isn't a bug. It's the product brief.

Most platforms treat Africa as an afterthought:
✗ No offline-first design
✗ No local payment integration
✗ No understanding of how commerce actually moves on the ground

I'm not building FOR Africa from outside.
I'm building FROM INSIDE — which means I design around constraints, not past them.

A restaurant manager in Cotonou shouldn't need a laptop farm to accept payments.
A migrant shouldn't need a tech degree to understand their case status.
An SME shouldn't need venture capital to have a professional online presence.

When you build from inside the constraint,
you don't create solutions for Africa.
You create products the world will eventually follow.

"The best opportunities aren't reserved for the right backgrounds.
They're waiting for the right conviction."

🌍 If you're young. African. Tech-driven. Apply anyway.
The world is watching. And it's building here.
```
